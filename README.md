# Travel Explorer 

Travel Explorer is a travel guide web application that helps users explore popular tourist destinations across India. The application displays information about places, travel details, and estimated tour prices.

## Live Demo

https://travel-explorer-nextjs-nbamkbvvx-sharmiladevelopers-projects.vercel.app/

## Features

* Explore famous tourist places in India
* View travel information and tour price estimates
* Clean and responsive user interface
* Fast performance using Next.js
* Easy navigation between destinations

## Tech Stack

* Next.js
* React
* TypeScript
* CSS / Tailwind

## Project Setup

Clone the repository:

git clone https://github.com/sharmilaDeveloper/travel-explorer-nextjs.git


Install dependencies:

npm install


Run the development server with the below command:

npm run dev


Open in browser with the below url:

http://localhost:3000
